1. ES6
1) 화살표 함수(Arrow Function)
   ES6 화살표 함수는 기존 함수 표현식에 비해 간결하다
   함수를 정의하는 영역의 this를 그대로 전달받을 수 있다

2) 자바스크립트의 this
   메소드, 함수가 호출될 때마다 현재 호출중인 메소드를 보유한 객체가 this로 연결된다
   현재 호출중인 메소드를 보유한 객체가 없다면 전역객체(Global Object, 브라우저 환경에서는
   window 객체)가 연결된다

   bind() : 함수의 this 값을 특정 객체로 고정시키는 역할을 한다
      지정할 객체를 this로 미리 연결한(binding) 새로운 함수를 리턴한다
   apply(), call() : 함수를 호출하면서 특정 객체를 this로 지정할 때 사용한다
      apply()메소드는 인수를 배열형태로 전달
      call()메소는 인수를 각각 직접 전달한다

3) 화살표 함수에서 this
	일반함수와 화살표 함수에서의 this차이는 함수가 중첩됩 떄 확인 가능하다
	
4) 새로운 객체 리터럴
	ES6에서 객체의 속성표기법을 개선
	객체의 속성을 작성할 때 변수명과 동일하게 한다면 생략이 가능
	객체 리터럴의 단축 표기법이라고 함
