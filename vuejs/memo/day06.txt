복습 -> 기본디렉티브 -> 컴포넌트 

0. 복습
vue
	user Interface 개발을 위한 자바스크립트 프레임워크
	html, css, javascript를 기반으로 사용자인터페이스ui를 효율적으로 개발할 수 있다
	v-접두어 붙은 특수속성을 디렉티브라고 함
	
1. 디렉티브directive
1) 바인딩(v-bind)
	데이터(상태) 속성에 바인딩할 때 사용하는 특수 속성임
	바인딩된 DOM은 Vue인스턴스의 속성으로 최신상태를 유지함(반응형동작)
	단방향으로만 데이터 바인딩을 수행함
	
2) 이벤트 핸들링(v-on)
	사용자가 앱과 상호작용할 수 있게 하기 위한 v-on디렉티브
	vue 인스턴스의 메소드를 호출하는 이벤트리스너를 추가할 수 있음
	
3) v-model 디렉티브
	vue는 양식(input, select 등)의 입력과 앱의 상태(data)를 양방향 바인딩할 수 있음
	사용자로부터 입력값을 갖고 싶은 경우 양방향 바인딩을 사용한다
	
수식어modifier
	디렉티브에 특별한 기능을 추가하는 vue.js 디랙타브 문법이다
	- lazy : 압력 폼에서 다른 요소로 포커스가 이동하는 이벤트가 발생할떄 
	압력한 값을 데이터와 동기화
	- number  : 숫자가 입력될 경우 number 타입의 값으로 자동형변환되어 데이터 옵션값으로 반영
 	- trim : 문자열의 앞뒤 공백을 자동으로 제거

V-model 한글처리문제
	값받는 시점이 1글자 입력완료될때이며 그때 처리됨
	이벤트 처리를 통해 해결가능함

4) v-if, v-show 디렉티브
	조건부 렌더링 디렉티브 
	지정한 조건에 따라서 html요소를 화면에 표현할지 말지 결정함

V-show : 화면에 보일지말지 결정함
	html요소는 생성하지만 화면에 보여주지 않을 수 있음
V-if : 조건에 부합하지 않으면 렌더링 수행

5) v-for 디렉티브
	반복적인 데이터를 처리할 때 사용
	for 구문은 원본데이터가 어떤 형식인가에 따라서 사용방법이 다름
	- 배열인 경우, 배열에서 하나씩 데이터를 꺼내서 변수에 할당하면서 반복
	- 객체인 경우, 키를 이용해 값에 접근하는 해시맵(hashMap)
		<key, value> 값을 얻어낼 수 있는 구조를 사용함
